Q1. 다음 중 객체지향 프로그래밍의 특징이 아닌 것은?
정답
캡슐화 상속 컴파일 다형성
해설

컴파일은 객체지향의 특징이 아닙니다. 객체지향의 4대 특징은 캡슐화, 상속, 다형성, 추상화입니다.

Q2. HTTP 상태 코드 404의 의미는?
오답
서버 오류 리소스 없음 권한 없음
내 답변: 서버 오류    정답: 리소스 없음
해설

404는 Not Found로 요청한 리소스를 찾을 수 없음을 의미합니다. 서버 오류는 5xx 계열 상태 코드입니다.

Q3. RESTful API의 특징으로 옳은 것을 모두 고르시오.
부분 정답 (5/10)
Stateless URI로 리소스 식별 SOAP 기반
내 답변: Stateless만 선택
해설

REST의 주요 특징은 Stateless와 URI 리소스 식별입니다. SOAP는 별도의 프로토콜로 REST와는 다릅니다.
