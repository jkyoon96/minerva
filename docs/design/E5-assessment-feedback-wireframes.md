```
[정답]      → 녹색 배경, ✓ 아이콘
[오답]      → 빨간색 배경, ✗ 아이콘
```

### AI 채점 신뢰도 상태
```
[높음 (85%+)]   → 초록색 배지, 자동 확정 가능
[보통 (70-85%)] → 노란색 배지, 검토 권장
[낮음 (<70%)]   → 빨간색 배지, 필수 검토
```

### 테스트 케이스 상태
```
[PASS]           → 녹색, ✓ 아이콘
[FAIL]           → 빨간색, ✗ 아이콘
[TLE]            → 노란색, 시간 초과
[MLE]            → 보라색, 메모리 초과
[Runtime Error]  → 주황색, 런타임 오류
```

### 표절 검사 상태
```
[정상 (0-20%)]   → 초록색 표시
[주의 (20-50%)]  → 노란색 표시
[의심 (50%+)]    → 빨간색 표시, 상세 검토 필요
```
